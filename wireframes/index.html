Website Reflection

The core focus of this website is to engage with theoretical frameworks and understand how data visualization constructs meaningful stories about information gathered. In its current state, the website lacks elements and characteristics which facilitate communication of the overall theme of data visualization. The future development of this website will be more focused on illustrating the communicative values of data visualization. This will be achieved by making the content of the website more interactive as a whole. The goal is creating a unique experience for the user and having them feel satisfied with the environment established by the website. Moving forward, Further development of the theoretical and technical content will be prioritized.
